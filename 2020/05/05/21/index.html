微软教育版A3订阅是不是钓鱼？

2020-05-05- 2022-07-29
1.9k- 7m
解答

微软自己的锅，配置错误。本订阅会在六月末撤销；本文旨在记录一个伟大订阅的诞生。

诸位只要对Office的事情关注一下，就知道现在忽然出来一阵撸教育版免费A3的风；有一部分人裤子都来不及穿就去薅微软羊毛了，而有一部分人则担心这是否是微软在钓鱼。
首先，要知道微软是否在钓鱼，我们就要弄清楚A3是什么订阅，这次的免费A3又是什么订阅。

Microsoft 365 A3

以下的英文部分是我摘抄自微软官网，中文部分来自谷歌翻译；其中，Microsoft 365 A3 for students HK$33.00/用户/月、HK$396.00/用户/年；Microsoft 365 A3 for faculty HK$45.00/用户/月、HK$540.00/用户/年；显然，此次的免费A3绝对不是这个订阅。

Microsoft 365 A3 for students

Office 365 A3, Enterprise Mobility + Security E3, Windows 10 Education E3, and Minecraft: Education Edition. This per-user licensed suite of Education products offers users best-in-class productivity across devices while providing IT security and control.

Office 365 A3，企业移动性和安全性E3，Windows 10教育版E3和Minecraft教育版。 此按用户许可的教育产品套件可为用户提供跨设备的一流生产力，同时提供IT安全性和控制力。

Microsoft 365 A3 for faculty

Office 365 A3, Enterprise Mobility + Security E3, Windows 10 Education E3, and Minecraft: Education Edition. This per-user licensed suite of Education products offers users best-in-class productivity across devices while providing IT security and control.

Office 365 A3，企业移动性和安全性E3，Windows 10教育版E3和Minecraft教育版。 此按用户许可的教育产品套件可为用户提供跨设备的一流生产力，同时提供IT安全性和控制力。

此次的免费教育版A3订阅

此次的免费教育版A3全称为：Microsoft 365 A3 - Unattended License for students use benefit，微软官方翻译是Microsoft 365 A3 - 适用于学生使用权益的无人参与许可

Microsoft 365 A3 - Unattended License for students use benefit

上图的一百五十万用户订阅就是此次的免费教育版A3许可证；

需要注意的是，此次A3订阅用户的数量一定程度上要和A1相匹配，比如我手里有一个一百五十万许可证的A1订阅，那么我最多只弄一百五十万许可证的A3订阅，多了可能会被微软风控。

但是也有大佬填一亿许可证，据说直到今早上午才翻车，存活时间超过了24小时

此订阅的特点

玄学，没错，微软最近的产品都挺玄学，从E5开发者到A3学生免费使用许可。

如此玄学：

  1. 早上购买此订阅被风控的概率比下午低；
  2. 存活时间再创记录，一小时之内被撤销许可的大有人在；
  3. 二零二零年五月四日是存活率最高的一天，往后的存活率基本是越来越低；
  4. 有缘人（有钱人）专属订阅；
  5. 订阅的许可数量其实并不影响风控概率（？）

此次免费订阅带来的影响

可以说非常巨大；E3 MSDN绝版只是前奏，此次的A3许可带来的影响比E3绝版带来的影响更加巨大，因为E3 MSDN绝版也仅仅是拉高了自己的身价，这次A3订阅直接把国际版Office 365黑市场重新洗牌了，下面我列举几个：

  1. 由于A3许可不仅存活率比E3（没错，就是那个350万订阅专坑小白的）高，而且在功能上完爆A1 PLUS（A3=Office 365桌面版+Win10企业版+我的世界教育版），导致A1 PLUS的价格暴跌，目前市场价在10至20元之间（子号），全局是可以订阅A3的，目前又升了（存疑？）
  2. 原来爹不疼妈不爱、鸡肋般的A1全局价格飙升，一个小时内狂翻七倍有余，从市场价50至300元之间升到了1200至5000元之间，目前仅有翻车A1（A3订阅被撤销的A1订阅，已被风控）有售（有传闻说A1全局要绝版，这也是A1全局价格飙升的原因之一）
  3. E3 MSDN这个香饽饽瞬间不香了，但目前价格还是偏高
  4. 现在有人出售A3子号了，由于暂不确定是否稳定，价格在20至100之间
  5. 有一大批人因此发家致富，2020年度最佳理财产品

综上所述，E3绝版仅仅是它自己的身价高了，而此次A3的横空出世，直接使A1 PLUS子号价格狂跌、A1全局价格暴涨、动摇了E3 MSDN的根基、养活了一大批仓鼠佬、E5开发者也变得无人问津

手里有A1 PLUS的可以当收藏用了，那可是曾经的性价比之王

推荐？

暂不，因为出来的时间太短了，某些坐等A3许可被撤销的可以收一批低价A1 P全局；现在有两类人：一类是裤子都不顾的去撸A3，一类趁机低价收翻车A1、A1 PLUS全局，最后鹿死谁手，不得定论。

钓鱼？

暂不认为是钓鱼，因为此订阅在使用订阅链接购买前，购买面板上无此订阅的显示（截至2020年05月03日）；且有电子发票存在
