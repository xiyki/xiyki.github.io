公共教学用电脑（多媒体）软件推荐

2020-10-01- 2021-05-03
2.4k- 8m
经验

原定2020年的文章，还是推到了2021。

公共教学用电脑在软件方面是异常重要的，教师需要使用软件教学，学生则需要从软件演示中接受教育；但是目前互联网上的软件推荐类文章，不仅极少有涉及到教育方面的，而且推荐的软件也是那些小圈子中的，简而言之，此类文章基本对教育教学活动较少有参考价值。
本人多年涉及此方面的一些实际应用，就此文章来浅谈一些自己的想法和建议。

文章的宗旨是以最常见或是较常见的软件作为推荐对象，以最简单的方式，达到我们的教育教学目的；面向读者是初步接触过电脑的学生、教师。
注意：无论在何种情况中，都不应在未得到教师/上级准许下私自为公用教学用电脑安装任何软件或是更换电脑系统！

理想的情况是（也是本文的假象情景）：学校给每间教室都配备有多媒体（公用教学用电脑），硬件配置不是很好，大致的硬件参考如下（本文章基于此硬件标准推荐软件）

对于软件情况而言，电脑系统应为Windows系统，也可以是其他Linux系统，最好可运行exe程序

系统

对于系统方面，文章会尽量的简洁扼要，不过多赘述。

Windows

Windows作为最常见的电脑桌面操作系统，因而针对Windows系统的计算机病毒比其他操作系统多出许多；对于Windows系统，漏洞补丁尽量及时安装，并安装一些安全软件。

Windows XP

Windows XP确实是迄今为止最稳定的Windows系统之一，即使有学校依然在使用XP系统也不是意料之外。对于XP系统而言，最大的难题是软件兼容性，所以本文也适当考虑的Windows XP的实际需求，尽量不会推荐有兼容性问题的软件。
Windows XP对电脑各方面的要求算是比较低的，所以一般硬件情况比较差的电脑（特别是古董机）还是首推Win XP。即便如此，此类硬件配置的电脑，应由学校统一更换新设备；电脑硬件老化不仅会影响教育教学质量，并且会在安全方面埋下种种隐患。

Windows 7

Windows 7是大部分公共教学用电脑首选操作系统，同时Windows 7也是最容易感染计算机病毒的操作系统；这就得扯一下Win XP，XP系统对于如今的众多计算机病毒都无法兼容，所以实际上受到病毒影响较大的是Windows 7。
Windows 7适合机械硬盘，不需要固态硬盘，这是众多学校将其作为首选操作系统的原因之一——固态硬盘比机械硬盘的价格要贵上一些；虽说如今固态硬盘的价格持续走低，但奈何数量巨大，采用机械硬盘诚然可以达到最高性价比。

Windows 10

Windows 10对于硬盘要求较高；但即便是固态硬盘，在其余硬件条件不好的情况下，依然不推荐安装Windows 10作为教学用系统；目前针对Windows 10的计算机病毒日渐增多，安全软件的选择迫在眉睫。

Linux

Linux作为一款优秀的开源操作系统，其分支众多，不乏某些奇葩学校会安装Linux系统，在此种情形下，需要使用者有一定的Linux操作基础，如果不想更换操作系统，推荐此类学校为教师提供Linux系统学习平台。
Linux的口碑在程序员的小圈子中还是很好的，但仍然不推荐没有Linux操作基础的人使用Linux作为主力操作系统。

其他

如果贵校有定制操作系统，希望此类学校可以向生产厂家索要详细说明书，并教导教师使用。

安全防护软件

本小节主要针对Windows系统，Linux等其他系统可以忽略此小节。

火绒

火绒仅对一些 PUA/PUF、恶意加驱锁主页的木马病毒的查杀方面相比于其他杀毒软件来说较强一些；其他方面还是有缺陷的，所以火绒实际定位是一款弹窗广告拦截软件。
依然推荐安装，可以和360搭配使用；如果不需要其杀毒功能，可以下载火绒拦截广告独立版。
火绒官网

360

360诟病已久，主要问题就是全家桶捆绑安装和各种弹窗广告层出不穷，但其杀毒能力毋庸置疑；如想安装360，可以先安装火绒；如此一来，火绒负责拦截360的弹窗广告和捆版安装，360只需负责杀毒即可。
360国际版同样值得一试，没有广告弹窗以及全家桶。
360中国版官网
360国际版官网

其他

学校如果有其他订阅付费软件，可以优先使用学校安装的软件。

文档、幻灯片演示软件

请安装正版软件！本文不提供盗版。

WPS

WPS几乎可以满足教学需求，并且足够轻量，快速，适合低配电脑；更为重要的是WPS提供免费的校园版，可以一用；与360相同，WPS也有中国版和国际版之分，而WPS国际版不支持中文，需要手动汉化，可参考这篇文章WPS国际版设置中文的方法（无脑汉化2019.08.25）。

要说明的一点是：我本人是Office的忠诚用户，但在公共教学用电脑软件方面，依然推荐WPS。
WPS中国（中国大陆地区）
WPS国际（除中国大陆地区）

Office

Office过于臃肿，仅且只有在教授office套件使用方法时需要，在其他情况下，不推荐使用office；教育/非盈利组织可以联系微软官方取得教育/非盈利优惠，购买正版Microsoft 365。

中文输入法

为什么要强调中文？因为英文可以不安装输入法。

微软Windows原生输入法

众里寻他千百度，蓦然回首，那人却在，灯火阑珊处。

尝试过千千万万种输入法，最后还是回归到了原生输入法。

手心输入法

手心输入法许久没有更新了，词库也没有其他输入法大，但对于教学活动来说，绰绰有余。

小狼毫输入法

小狼毫输入法是一款开源输入法，这就意味着它不以“盈利”为目的；诚然，这款输入法并不适合中国大陆地区的人使用，使用习惯上更适合台湾、香港地区的人使用；如何切换为简体可以参考：小狼毫输入法如何设置成简体？。

其他

输入法因人而异，我推荐的上述三个软件，只是我认为比较好用的软件，有其他意见的可以发在评论区。

音频播放软件

配合教师的教学活动。

一众国产软件

酷狗、QQ音乐、网易云音乐之类的，条件允许的话可以全部安装；毕竟老师常用的音乐软件不一样，如果不全部安装，音乐格式将是一个大问题。

Pot Player

Pot Player当然是个很好的软件，如果不了解可以忽略此软件；官网在外网，中国大陆地区的IP连不上属正常现象。
Pot Player官网

解压/压缩软件

有些老师会使用在网络上下载的资源，所以解压软件也是必不可少的了。

360 压缩

360压缩确实不错，足够轻量、简洁，符合我们的需求；以360的尿性，360压缩国际版也是和360杀毒国际版一样，没有了中国版的花里胡哨；界面也是颇具欧美简约风（其实就是简陋）。
360压缩
360zip

Bandizip

Bandizip是韩国出产的软件，网络上许多文章是吹它的，实际上功能和win zip差不多，好处是免费，但最近有它想改为收费软件的消息。
Bandizip

7-zip

7-zip是一款开源软件，操作起来比较复杂，但轻量简洁。
典型的欧美风（简陋）网站——7-zip官网
